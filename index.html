Frequently Asked Questions

What is CI_τ measuring exactly?

CI_τ measures internal model coherence—how well an agent's predictions match reality, adjusted for energy efficiency, stability, and information density. It's a weighted metric (α·Σ + β·M + γ·Q + δ·Ω + ε·E + ζ·A) for agent learning quality.

Does this system create conscious AI?

No. We measure internal model coherence using thermodynamic and information-theoretic principles. We do not claim to measure subjective experience, sentience, or consciousness. CI_τ is a metric for agent learning quality and model stability.

Can I reproduce experiments?

Yes. Same agent seed → same CI curve. Deterministic reality feed, complete JSON logs, and exportable agent states ensure full reproducibility.

What's the energy measurement?

Energy values (0.01–0.05) are normalized dimensionless units scaled to Landauer's thermodynamic minimum (2.87e-21 J/bit), not raw Joules. They're efficiency proxies.
